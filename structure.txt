📦 4. Core GitHub Structure (VERY IMPORTANT)
4.1 Repository (Repo)

A repository is a project folder.

Example:

ecommerce-backend/
social-media-api/
portfolio-site/

It contains:

code
history
configuration
documentation
4.2 Commit

A commit = saved snapshot of code

Example:

“Added login API”

Each commit has:

unique ID (hash)
author
timestamp
changes
